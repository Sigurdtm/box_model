Main i HTML

Main er hvor man har innholdet av dokumentet. Kan ikke være mer enn en main.

Farging med hex-verdi

Velg hva du vil farge med f.eks background-color: Skriv # og 3-6 tall med verdien 0-9, eller bokstavene a-f. 0 er mørkest og f er lysest f.eks: #ffffff (hvit), #000000 (svart)

Farging med rgb-verdi

Skriv rgb(96, 168, 168) (verdien til fargen i parantes)

Hva brukes flexbox i CSS til

Flexbox gjør det lettere å designe layout på nettsiden. Mer effektiv måte å plassere “Align” osv. Funker bra selv om størrelser er ukjente og/eller dynamisk.
